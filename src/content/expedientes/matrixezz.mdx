## Resumen Técnico

Herramienta desarrollada para un entorno académico real, capaz de resolver teoremas matemáticos avanzados (Gauss, Gauss-Jordan, determinantes, inversa). Delega el cálculo algorítmico al backend Java para garantizar precisión numérica, manteniendo el cliente ligero con HTML, SCSS y JavaScript vanilla.
